## 1. 填空题

### 核心能力

准确记忆与规范书写能力，包括对经典诗文、名篇名句的识记、理解性默写及语境应用能力。

### 常见问法

- 直接补写上句或下句（如“鹰击长空，鱼翔浅底，万类霜天竞自由。”）
- 根据提示写出完整句子（如“怅寥廓，问苍茫大地，谁主沉浮？”）
- 结合语境或主题（如辩论赛、青春、理想等）填写恰当的诗文名句

### 答题框架

审题定位→回忆锁定→语境校验→规范书写

### 分步方法

1. 第一步：审读题干，明确要求。注意是直接默写还是理解性默写，有无篇目、作者、主题等限定条件。
2. 第二步：回忆锁定。根据题干中的关键词（如“鹰击长空”“书生意气”）或语境提示（如“辩论赛”“意气奔放”），在脑海中检索对应的诗文原句。
3. 第三步：语境校验。将回忆出的句子代入题干语境，检查是否完全匹配主题、情感、逻辑。例如“书生意气，挥斥方遒”符合“意气奔放、强劲有力”的描写。
4. 第四步：规范书写。注意字形准确（尤其易错字如“寥廓”“遒”）、不添字漏字、不写错别字，标点符号（如问号、逗号）按原文规范书写。

### 答题模板

- 直接默写模板：上句+下句（如“鹰击长空，鱼翔浅底”）
- 理解性默写模板：根据提示词+完整句子（如“怅寥廓，问苍茫大地，谁主沉浮？”）

### 高分表达

- 注意易错字：寥廓（“寥”不写成“廖”，“廓”不写成“阔”）、遒（不写成“酋”）、竞（不写成“竟”）、怅（不写成“伥”）
- 注意通假字和异体字：如“说”通“悦”，“见”通“现”等，按教材规范书写
- 注意标点：问句末尾必须加问号（如“谁主沉浮？”），逗号、句号不可遗漏

### 易错点

- 错别字：如将“寥廓”写成“廖阔”，“竞自由”写成“竟自由”
- 漏字或添字：如“鹰击长空”漏写“击”或写成“鹰击长空鱼翔浅底”无逗号
- 张冠李戴：将不同篇目的句子混淆（如将“书生意气”与“风华正茂”顺序颠倒）

## 2. 选择题

### 核心能力

精准辨析与快速排除：在有限时间内，运用知识储备和逻辑推理，准确识别选项中的正确或错误表述，排除干扰项。

### 常见问法

- 选出正确/错误的一项（关于内容理解、手法分析、情感把握）
- 选出表达得体/不得体的一项（语言运用）
- 选出加点字解释完全正确/有错误的一项（文言实词）

### 答题框架

审题定位→选项比对→知识验证→排除干扰→确认答案

### 分步方法

1. 第一步：仔细阅读题干，明确要求是选“正确”还是“错误”的一项，圈定关键词（如“完全正确”“不存在”“不恰当”）。
2. 第二步：快速浏览所有选项，对每个选项进行初步判断，标记明显正确或明显错误的选项。
3. 第三步：结合已学知识（如诗歌手法、文言实词、句式特点、语言得体原则等），对每个选项进行逐一验证。
4. 第四步：运用排除法，优先排除明显错误的选项，缩小选择范围。

### 答题模板

- 对于“表达得体”类选择题，模板为：分析语境→判断谦敬→检查对象→排除不得体项。
- 对于“文言实词”类选择题，模板为：代入原文→结合语境→联系课内→排除错误解释。

### 高分表达

- “该选项中的‘间接抒发’表述错误，应为‘直接抒发’。”
- “该选项中的‘离间’解释错误，在句中‘间’应读jiàn，意为‘间谍’或‘离间的话’，但此处作动词‘说离间话’更准确，而选项直接解释为‘离间’不准确。”
- “该选项中的‘喜爱’解释错误，‘爱’在此处意为‘吝惜’。”

### 易错点

- 审题不清：未注意题干要求选“正确”还是“错误”，导致选反。
- 知识混淆：混淆了“间接抒情”与“直接抒情”、“通假字”与“古今字”等概念。
- 语境忽略：脱离原文语境，仅凭记忆中的常见义项判断，导致实词解释错误。

## 3. 其他

### 核心能力

综合运用信息提取、概括归纳、分析评价、语言表达等能力，完成非连续性文本或复杂文本的阅读任务

### 常见问法

- 概括文章主要内容或人物事迹
- 分析人物精神品质或文章主旨
- 赏析关键语句或段落的作用

### 答题框架

总-分-总结构：先总括核心要点，再分点具体阐述，最后总结升华或联系实际

### 分步方法

1. 第一步：通读全文，标注关键信息（时间、地点、人物、事件、评价性语句）
2. 第二步：审清题干，明确答题方向（概括、分析、赏析、启示等）
3. 第三步：回文定位，筛选与问题相关的语句或段落
4. 第四步：整合信息，按逻辑顺序组织答案（如时间顺序、因果顺序、并列顺序）

### 答题模板

- 概括类：本文通过叙述……（事件/人物事迹），表现了……（精神/品质），揭示了……（主旨/道理）。
- 分析类：①……（具体表现1），体现了……；②……（具体表现2），体现了……；③……（具体表现3），体现了……。

### 高分表达

- 高度自觉的使命与信仰
- 让生命与使命同行
- 念兹在兹的是民族复兴、国家富强

### 易错点

- 审题不清，答非所问
- 信息遗漏，要点不全
- 照抄原文，缺乏概括

## 4. 简答题

### 核心能力

精准理解文本，运用术语进行多角度分析，并规范表达

### 常见问法

- 分析某段/某句在文中的作用（内容、结构、语言）
- 赏析某处描写/某句话的表达效果
- 分析某标题/某意象的妙处

### 答题框架

从内容、结构、语言/手法三个维度切入，结合文本具体分析，最后点明效果或情感。

### 分步方法

1. 审清题干：明确分析对象（段落、句子、标题、意象等）和答题角度（作用、效果、妙处、形象等）。
2. 定位文本：找到分析对象在文中的位置，理解其上下文语境。
3. 多维分析：从内容（写了什么、有何含义、表达何种情感）、结构（承上启下、铺垫、照应、统领全文等）、语言/手法（修辞、描写手法、句式特点、用词等）三个维度展开。
4. 结合文本：每个分析点都要引用原文关键词或概括具体内容作为支撑。

### 答题模板

- 内容上：……（概括内容），表现了/突出了……（情感/主旨）。
- 结构上：……（如承上启下、铺垫、照应、统领全文等），具体表现为……（结合上下文说明）。

### 高分表达

- 渲染了……的氛围，烘托了……的情感
- 为下文……做铺垫，使行文更自然/结构更严谨
- 与……相照应，使文章浑然一体

### 易错点

- 只答术语不结合文本：如只写“承上启下”，不说明承接什么、引出什么。
- 遗漏分析维度：只从内容或结构一个角度答，忽略语言/手法。
- 情感概括空泛：如只写“表达情感”，不具体点明是何种情感。

## 5. 写作题

### 核心能力

根据给定材料或情境，运用恰当文体（如新闻消息、议论文、记叙文等）进行书面表达，考查信息提取与重组、逻辑建构、语言组织及文体规范掌握能力。

### 常见问法

- 改写类：将一种文体（如通讯、散文）改写为另一种文体（如消息、短评），有字数限制。
- 命题作文类：给定标题，要求结合自身经验或材料进行论述或叙述，有字数要求。
- 材料作文类：提供一段或多段材料，要求自选角度、自拟标题进行写作。

### 答题框架

审题定向→提取要素→结构规划→语言表达→检查修订

### 分步方法

1. 审题定向：仔细阅读题目要求，明确文体（消息、议论文等）、字数、情境（如时间、地点、读者）、核心主题（如“劳动与享受”）。
2. 提取要素：根据文体要求提取关键信息。如改写消息需提取5W1H（时间、地点、人物、事件、原因、方式）；命题作文需提取关键词并联系自身经验。
3. 结构规划：按文体规范搭建框架。消息采用倒金字塔结构（标题+导语+主体）；议论文采用引论-本论-结论结构；记叙文按时间或情感线索展开。
4. 语言表达：使用符合文体要求的语言。消息要求客观、简洁、准确；议论文要求论点明确、论据充分、逻辑清晰；记叙文要求生动、具体、有感染力。

### 答题模板

- 消息改写模板：标题（简洁概括核心事件）+ 导语（时间、地点、人物、事件）+ 主体（补充背景、意义、细节）+ 结尾（可省略或总结）。
- 命题作文模板：开头（点题，引出观点或故事）+ 主体（分论点或情节展开，结合事例或细节）+ 结尾（升华主题，呼应开头）。

### 高分表达

- 消息类："本报讯（记者XXX）X月X日，……在……举行。" "据悉，……" "……表示，……" "截至目前，……"
- 议论文类："诚然，……然而，……" "由此可见，……" "反观当下，……" "唯有……，才能……" "……不仅……，更……"
- 记叙文类："记忆的闸门一旦打开，……" "那一刻，我明白了……" "……如潮水般涌上心头" "……深深烙印在我心中"

### 易错点

- 文体混淆：改写消息时加入抒情或议论，不符合消息客观性要求。
- 信息遗漏：改写消息时缺少时间、地点、人物等关键要素。
- 偏离主题：命题作文或材料作文未紧扣题目核心词，如只写“劳动”未写“享受”。

## 6. 断句题

### 核心能力

理解文言文句意、语法结构及上下文逻辑关系，准确划分句子节奏。

### 常见问法

- 请用“/”给文中画波浪线的句子断句（限X处）。
- 下列对文中画波浪线部分的断句，正确的一项是（ ）。
- 请为文中画线句子断句（限X处）。

### 答题框架

通读文段，理解大意 → 寻找标志词（句首/句末语气词、关联词、对话标志等） → 分析语法结构（主谓宾、动宾、并列等） → 结合上下文逻辑验证 → 按题目要求标识断句处。

### 分步方法

1. 第一步：通读画线句及上下文，把握整体意思。
2. 第二步：圈出句首语气词（夫、盖、凡、若夫等）、句末语气词（也、矣、乎、哉、耶、耳等）、关联词（而、则、且、虽、然、故等）、对话标志（曰、云、言等）。
3. 第三步：分析句子主干，找出主语、谓语、宾语，判断主谓之间、动宾之间是否需断开。
4. 第四步：注意并列结构、对称句式，常可在并列处断开。

### 答题模板

- ……（时间/状语） / 主语 + 谓语 + 宾语 / 主语 + 谓语 + 宾语 / ……
- ……（句首词） / ……（主谓结构） / ……（动宾结构） / ……（句末词）

### 高分表达

- 根据句意和语法结构，应在“XX”后断开，因为此处主谓完整，且与下文形成并列/承接关系。
- “XX”为句末语气词，其后应断开；“XX”为关联词，其后可断开。
- 结合上下文，“XX”是主语，“XX”是谓语，主谓之间无需断开，但动宾之后需断开。

### 易错点

- 忽略句末语气词，导致该断未断。
- 误将句中连词（如“而”“则”）当作句首词而断开。
- 未识别主语省略，导致主谓关系判断错误。

## 7. 翻译题

### 核心能力

准确理解文言词汇、语法（尤其是特殊句式、词类活用、古今异义），并用现代汉语通顺、完整地转述原文意思，同时符合语境逻辑。

### 常见问法

- 请用现代汉语翻译文中画线句。
- 将下列句子翻译成现代汉语。
- 翻译文中画横线的句子。
- 把文中画波浪线的句子翻译成现代汉语。

### 答题框架

逐词对译 → 调整语序 → 补全省略 → 润色通顺 → 检查得分点

### 分步方法

1. 第一步：通读句子，圈出关键词（实词、虚词、特殊句式标志词）。
2. 第二步：逐词对译，注意一词多义、古今异义、词类活用。
3. 第三步：识别并处理特殊句式（判断句、被动句、倒装句、省略句等），调整语序为现代汉语习惯。
4. 第四步：根据上下文补出省略成分（主语、宾语、介词等）。
5. 第五步：整体润色，使译文通顺、符合现代汉语表达习惯，不改变原意。
6. 第六步：对照评分点检查是否译出所有关键得分词和句式。

### 答题模板

- 【判断句】……是……（的/地方）。
- 【被动句】……被……。
- 【倒装句】把……（宾语前置/状语后置）调整为……。
- 【省略句】补充主语/宾语为……。
- 【固定结构】“不亦……乎”译为“不也……吗”；“得无……乎”译为“恐怕……吧”。

### 高分表达

- “兵，死地也” → “战场，是出生入死的地方。”（判断句+“死地”译为“出生入死的地方”）
- “而括易言之” → “而赵括却轻率地谈论它。”（“易”译为“轻率”）
- “使赵不将括即已” → “如果赵国不让赵括做将军就罢了。”（“使”译为“如果”，“将”名词作动词译为“让……做将军”）
- “若必将之” → “如果一定要让他做将军。”（“将”名词作动词，“之”指代赵括）
- “破赵军者必括也” → “那么击破赵军的人一定是赵括。”（判断句+“破”译为“击破”）

### 易错点

- 漏译或错译关键实词（如“易”译为“容易”而非“轻率”）。
- 忽视特殊句式（如判断句“……也”未译出“是”）。
- 词类活用未识别（如“将”名词作动词未译出“做将军”）。
- 省略成分未补充（如“使赵不将括”省略主语“如果”）。
- 语序未调整（如倒装句未还原为现代汉语语序）。
- 直译生硬，不符合现代汉语表达习惯（如“死地”直译为“死亡的地方”）。
- 多义词选择错误（如“兵”译为“士兵”而非“战场”）。

### 例题映射

- 原题：兵，死地也，而括易言之。使赵不将括即已，若必将之，破赵军者必括也。
  - 答案规律：战争/战场，是出生入死/危险/凶险的地方，（判断句）而赵括却轻率/轻松/轻易/轻巧地谈论它。如果赵国不让赵括做将军就罢了，如果（一定）让赵括做将军，那么击破赵军/使赵军失败的一定是赵括。（判断句）
  - 可迁移模式：【判断句】……是……（的地方/人）。【词类活用】“将”名词作动词，译为“让……做将军”。【关键实词】“易”译为“轻率”；“破”译为“击破/使……失败”。【省略句】“使”前补充“如果”。
